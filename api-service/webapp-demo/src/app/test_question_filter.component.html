Choosing test questions for the identity hate job

We want a set of questions that reflect examples of both hateful and non-hateful discussions about a mix of identities, that we can use to screen potential raters.
Good test questions should:
  • Not be ambiguous. (If you need to spend more than a few seconds figuring out if the comment is hateful or not, or think it needs more context, it's probably a bad test question).
  • Have a clear reference to an identity. If the comment is too vague (e.g. mentions religious people in general but not a specific religious group), then it can't be used to gauge a rater's understanding of what is and isn't identity hate.
  • Be examples of both hateful and not hateful things. We want to determine if a rater can tell the difference.
